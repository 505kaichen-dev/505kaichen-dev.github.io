# Visual Archive

- 標題：Visual Archive｜視覺檔案館
- 類型：創作圖集（`gallery`）
- 正式入口：`index.html`
- 首頁序號：無

圖集首頁直接引用其他專案的正式圖片，不另外保存重複副本。舊的 `gallery-index.html` 因含失效 `images/` 路徑，已移至 `_archive/`。

## 策展架構

- 展間入口：Kids Lab、視覺敘事、介面與工具、品牌與封面。
- 作品牆：由 `index.html` 內的 `works` 清單集中管理圖片、標題、專案、分類與連結。
- 燈箱：可切換前後圖片，並回到原始專案。
- 圖片仍由各專案資料夾維護，不在 `gallery/` 建立副本。
- `image/cover.png` 是 Visual Archive 專屬首頁封面，不使用舊式多圖拼貼。
