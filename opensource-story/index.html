// 緣起

一個關於慷慨與瘋狂的故事

想像一個人，花了好幾年時間，寫出一個足以改變整個產業的工具——然後選擇把它免費送出去，歡迎任何人使用、修改、再分發。

這不是一個人的故事，而是幾十年來無數工程師共同選擇的生活方式。他們相信：知識不應該被鎖起來。軟體就像數學，屬於全人類。

「自由軟體是一個自由的問題，不是價格的問題。把它想成是言論自由，而不是免費啤酒。」

— Richard Stallman，自由軟體運動之父，1983

正是這種近乎偏執的信念，催生了今天幾乎所有你使用的數位世界的基礎——從手機作業系統、網路伺服器，到 AI 模型訓練框架。

// 歷史時間軸

四十年，一條改變世界的線

從一個人的憤怒出發，到全球一億人的協作平台

自由軟體運動
1983
GNU 計畫誕生

Richard Stallman 因為無法修改一台印表機的驅動程式而憤怒，決定啟動 GNU 計畫，目標是建造一個完全自由的作業系統。他的宣言成為開源運動的精神基礎。

RS
Richard Stallman · MIT
授權框架
1989
GPL 授權條款問世

GNU 通用公共授權條款（GPL）正式發布，確立了「Copyleft」概念：你可以自由使用這個程式碼，但你修改後的版本也必須保持開放。這個法律框架讓開源運動有了制度保護。

作業系統革命
1991
Linux 核心誕生

芬蘭大學生 Linus Torvalds 在郵件列表上寫道：「我在做一個免費的作業系統……只是個人愛好，不會像 GNU 那麼大、那麼專業。」今天，全球 96% 的雲端伺服器跑在 Linux 上。

LT
Linus Torvalds · 赫爾辛基大學
網路基礎
1995
Apache、MySQL、PHP 先後開源

這一年是開源的爆炸年。Apache 網頁伺服器、MySQL 資料庫、PHP 語言相繼以開源形式發布，後來組合成「LAMP Stack」，驅動了早期 90% 的網際網路網站，包括維基百科與 WordPress。

版本控制
2005
Linus 用兩週寫出 Git

因為與商業版本控制工具 BitKeeper 鬧翻，Linus Torvalds 花了兩週自己寫了一個分散式版本控制系統——Git。他說這個工具是「為了讓我自己能繼續維護 Linux」。結果它改變了整個軟體開發世界。

LT
Linus Torvalds
平台誕生
2008
GitHub 上線，協作進入新時代

四位工程師創建了 GitHub，把 Git 的強大包裝成人人可用的平台。開源從此不再只是技術人的地下文化，而成為全球開發者每天生活的一部分。React、Kubernetes、TensorFlow 都在這裡誕生並成長。

企業收購
2018
微軟以 75 億美元收購 GitHub

開源社群一片譁然，怕微軟「污染」這塊淨土。結果出乎意料——微軟不但沒有關閉免費服務，還大幅擴展功能，並將自己大量產品開源。這成為商業力量與開源精神罕見和平共存的案例。

AI 時代
2021—今
開源程式碼成為 AI 的養分

幾十年來工程師無私分享的程式碼，成為訓練 GitHub Copilot、ChatGPT 等 AI 系統的原始材料。開源精神的遺產，正以意想不到的方式加速人類下一個科技浪潮。到 2025 年底，GitHub 年度營收已超過 30 億美元。

知識應該流動

數學公式不需要版權，物理定律不需要授權。在這群人眼中，軟體更接近數學而非商品——它屬於全人類的智識遺產。

∑

集體智慧更強

一個人寫的程式碼有限，但全世界工程師共同審視、改進、叠加的程式碼，能達到任何個人或公司都無法企及的高度。

↗

聲譽是真實回報

當你的程式碼被全球最頂尖的工程師使用與討論，你得到的同儕認可，對真正熱愛技術的人而言，有時比金錢更有價值。

⊕

生態系讓自己更強

你讓別人的工具更好，別人也在讓你的工具更好。開源是正和遊戲，不是零和競爭——整個生態系的成長讓每個人受益。

≠

問題本身才是重點

頂尖技術人對「解決問題」有近乎偏執的熱情。商業化意味著銷售、管理、法務——這些對他們不只是無聊，而是干擾。

∞

知識的複利效應

前人的程式碼讓後人站在更高的肩膀上，後人的成果再讓下一代起步更高。這種知識疊加的速度，在人類歷史上前所未有。

站在這些人肩膀上

他們不一定有商業頭腦，但每個人都改變了世界運作的方式。

Linus Torvalds

Linux · Git · 芬蘭

芬蘭大學生，用個人興趣寫出 Linux 核心，後來又因為不滿商業工具而花兩週造了 Git。今天全球 96% 雲端伺服器跑在他的作品上，他本人至今仍在審查每一個核心的 Pull Request。

「Talk is cheap. Show me the code.」
RS

Richard Stallman

GNU · GPL · MIT

因為無法修改一台印表機的驅動程式而憤怒，從此發起自由軟體運動。他創造的 GPL 授權框架，為整個開源世界提供了法律基礎，被稱為「軟體自由的守護者」，其偏執程度令人既敬畏又敬佩。

「自由軟體是言論自由，不是免費啤酒。」
GH

Geoffrey Hinton

深度學習之父 · 多倫多大學

在神經網路幾乎被學術界放棄的黑暗年代，獨自堅持研究數十年。2012 年 AlexNet 震驚世界，開啟了今天的 AI 浪潮。他後來加入 Google，又因擔憂 AI 風險而公開辭職發出警告。

「我花了四十年在研究一個大家都說沒用的東西。」
TBL

Tim Berners-Lee

WWW 發明者 · CERN

在 CERN 工作時發明了全球資訊網（WWW），並選擇將這項技術完全開放——沒有申請專利，沒有收取授權費。他說：「如果它是開放的，它就能成長。」今天每個人每天使用的網際網路，是他的禮物。

「這個網路是給所有人的。」
// 2018.02.28 · 資安考驗

當世界上最大的攻擊
打上門來

你可能以為 GitHub 從未被攻擊過。事實是——它遭受過人類歷史上最大規模的 DDoS 攻擊，只是因為應對太好，你沒有感覺到。

1.35 Tbps 攻擊流量峰值
1.27億 每秒封包數
8 分鐘 從攻擊到完全緩解
17:21 UTC
攻擊開始。駭客利用 Memcached 快取系統漏洞，產生 51,200 倍的流量放大效應，GitHub 網站開始無法存取。
17:23 UTC
GitHub 啟動緊急應對，聯繫 Akamai Prolexic DDoS 防護服務，將所有流量重新導向至清洗中心進行過濾。
17:30 UTC
攻擊停止。整個過程不到 10 分鐘，大多數用戶幾乎沒有察覺。GitHub 隨後公開發布完整事件報告。

這次攻擊成為網路安全教科書級別的案例。它揭示了一個真相：再強大的平台也會被攻擊，但真正的實力體現在被打倒後多快能站起來——以及多透明地告訴所有人發生了什麼。
