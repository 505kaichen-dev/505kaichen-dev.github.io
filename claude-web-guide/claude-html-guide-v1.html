Chapter 01

網頁到底是什麼？
三分鐘搞懂基礎概念

先了解幾個關鍵詞，後面讀起來會順很多。

HTML

網頁的「骨架」

決定一個網頁上有什麼東西——標題、段落、圖片、按鈕放在哪。就像蓋房子先打鋼筋水泥的框架。

CSS

網頁的「外觀」

決定字體顏色、背景色、間距、動畫效果……讓骨架變得好看。就像幫房子裝潢貼磁磚刷油漆。

JavaScript

網頁的「動作」

點按鈕後跳出視窗、輪播圖片、送出表單——這些互動行為都靠它。就像讓房子裡的電燈可以開關。

HTML5

HTML 的最新版本

現在所有網站預設都用 HTML5，支援影片、音訊、更豐富的互動功能。只要叫 Claude 做 HTML 就會自動用它。

💡
好消息是：你完全不需要自己寫這些東西。Claude 會幫你全部搞定，你只要用中文說清楚你想要什麼就好。
流程說明

Claude 怎麼幫你做網站？

整個過程超簡單，就四個步驟。

1

你用中文描述想要的網頁

告訴 Claude 網頁的用途、風格、內容——就像和朋友說話一樣，不需要任何技術語言。

2

Claude 自動生成完整的網頁程式碼

Claude 會把你說的話翻譯成 HTML + CSS + JavaScript，輸出一個完整的網頁檔案。

3

複製程式碼，存成 .html 檔案

把 Claude 輸出的內容複製起來，用「記事本」或任何文字編輯器貼上，儲存時把副檔名改成 .html。

4

用瀏覽器打開，就看到成品！

雙擊剛才存的 .html 檔案，Chrome 或 Edge 就會直接把它當網頁打開，你的網站就出現了。

Chapter 02

提示詞怎麼寫？

「提示詞」就是你對 Claude 說的話。寫得越清楚，網頁就越符合你的期待。

🔴 必填項目 — 一定要寫
🟢 選填項目 — 有才需要寫
必填 · 網頁風格
一定要寫
告訴 Claude 你想要什麼感覺的網頁。 // 範例寫法： 風格要現代、高質感、深色背景搭配紫色主色調。 // 或是： 風格要清爽、乾淨、白色背景、日系極簡風格。 // 或是參考某個網站： 風格參考 Apple 官網，要非常簡潔大器。
必填 · 網頁內容
一定要寫
說明網頁要放什麼資訊、用途是什麼。 // 範例：個人作品集 網頁是我的個人作品集，包含： ・自我介紹（我叫王小明，是平面設計師） ・作品展示區 ・聯絡方式（email: xxxx@gmail.com） // 範例：活動宣傳頁 這是一個咖啡廳的形象頁，要有： ・店名「晨光咖啡」 ・slogan「從一杯咖啡開始美好的一天」 ・營業時間：週一到週日 08:00–21:00 ・地址：台北市大安區
選填 · 參考網站或圖片
有的話再填
有沒有看到哪個網站讓你覺得「就是這種感覺！」？ 直接把連結貼給 Claude。 // 範例寫法： 請參考這個網站的排版風格： https://www.notion.so （把你想參考的網址貼在這裡） // 也可以截圖上傳給 Claude 看： （直接在對話框附上截圖） 「請做出類似這張圖的網頁風格。」
選填 · 廠商或品牌 LOGO
有的話再填
如果你有品牌 LOGO 圖檔，可以請 Claude 把它放進網頁。 // 重要：圖檔要和 HTML 檔案放在同一個資料夾 // 範例寫法： 請在網頁左上角放上公司 LOGO， 圖檔名稱是 logo.png（和 HTML 在同一個資料夾）。 // 如果是 SVG 格式： LOGO 檔案是 brand-logo.svg，請放在導覽列左側。
選填 · 要放入的圖片
有的話再填
有要放圖片的話，告訴 Claude 圖檔名稱和放在哪個位置。 // 圖檔也要和 HTML 放在同一個資料夾 // 範例： 請在頁面頂部放一張大圖， 圖檔名稱是 hero-banner.jpg。 作品展示區有三張圖，分別是： ・work-01.jpg — 放第一件作品 ・work-02.jpg — 放第二件作品 ・work-03.jpg — 放第三件作品
選填 · 動態效果與互動
有的話再填
想要特別的視覺效果可以直接說。 // 常見的效果： ・捲動時元素慢慢淡入出現 ・滑鼠移到按鈕上時有發光效果 ・頁面載入時有動畫 ・圖片 hover 時放大顯示 ・背景有微粒子飄動的效果 // 範例寫法： 請加入捲動時各區塊逐一淡入的動畫效果， 按鈕 hover 時要有柔和的發光感。
選填 · 語言與字型
有的話再填
想要指定語言、字型風格也可以說。 // 範例： 請使用繁體中文字型，正文字體大一點，讀起來輕鬆。 // 或是： 標題想要英文配中文的混搭風格。 // 或指定 Google Fonts 字型： 標題請用 Playfair Display 字型，內文用 Noto Sans TC。
選填 · 手機版（RWD 響應式設計）
建議加上
// RWD（Responsive Web Design）= 手機、平板、電腦都能好好顯示 現在大部分人用手機看網頁， 強烈建議在提示詞裡加上這句話： 「請確保網頁在手機、平板和電腦上都能正常顯示（RWD 響應式設計）。」

萬用提示詞範本

把下面這個範本複製過去，填入你的資訊，直接丟給 Claude 就搞定。

✦ 萬用提示詞範本
// ── 基本設定（一定要填）── 請幫我用 HTML5 製作一個單頁網頁。 【網頁用途】（填入：例如「個人作品集網站」、「咖啡廳形象頁」） 這個網頁是：_______________ 【風格】（填入你想要的視覺風格） 網頁風格要：_______________ （例如：深色系、極簡、現代、溫暖、科技感） 【主要內容】（列出網頁要放什麼） 內容包含： ・_______________ ・_______________ ・_______________ // ── 選填項目（有才填）── 【參考網站】（可貼網址或上傳截圖，沒有可刪除） 風格請參考：_______________ 【LOGO】（填入 LOGO 圖檔的完整檔名，沒有可刪除） 請放入品牌 LOGO，檔名為：_______________ 【圖片】（填入圖片的完整檔名，沒有可刪除） 請放入以下圖片： ・_______________（說明這張圖放在哪個位置） ・_______________ 【動態效果】（說明想要什麼效果，沒有可刪除） 請加入：_______________ // ── 固定設定（直接保留這段，不用改）── 請確保： ・網頁在手機、平板和電腦都能正常顯示（RWD 響應式設計） ・所有圖片和 HTML 檔案放在同一個資料夾 ・輸出完整可用的 HTML 檔案，CSS 和 JS 全部寫在同一個 .html 檔案裡
紅色 = 必填
綠色 = 選填（沒有就整行刪掉）
灰色 = 說明文字（可以直接保留）

填完後的完整提示詞範例
請幫我用 HTML5 製作一個單頁網頁。 【網頁用途】 這個網頁是：攝影師個人作品集網站 【風格】 網頁風格要：深色背景、高質感、電影感、以黑白灰為主搭配金色點綴 【主要內容】 內容包含： ・頁首：我的名字「陳大明」和職業「自由攝影師」 ・關於我：簡短自我介紹（3-4 句話，請幫我用英文寫個有質感的介紹） ・作品集區塊：放 6 張作品圖片的網格排版 ・聯絡我：Email → photo@example.com，Instagram → @da_photo 【參考網站】 風格請參考：https://www.awwwards.com 【LOGO】 （略，沒有 LOGO） 【圖片】 請放入以下圖片： ・work-01.jpg（作品集第一張） ・work-02.jpg（作品集第二張） ・work-03.jpg（作品集第三張） ・work-04.jpg（作品集第四張） ・work-05.jpg（作品集第五張） ・work-06.jpg（作品集第六張） 【動態效果】 請加入：捲動時作品圖片逐一出現的淡入效果， 圖片 hover 時有放大和顯示說明文字的效果 請確保： ・網頁在手機、平板和電腦都能正常顯示（RWD 響應式設計） ・所有圖片和 HTML 檔案放在同一個資料夾 ・輸出完整可用的 HTML 檔案，CSS 和 JS 全部寫在同一個 .html 檔案裡

不滿意結果？直接繼續跟 Claude 說！例如：「按鈕顏色改成藍色」、「字型再大一點」、「頁首的圖改成全寬滿版」——不需要重新開始，Claude 會幫你修改。
Chapter 04

做好了！怎麼看到網頁？

Claude 會給你一段程式碼，三種方法都能讓你馬上看到完成的網頁。

📄

方法一：存成 .html 檔案

複製 Claude 給你的程式碼 → 打開電腦的「記事本」(Windows) 或「文字編輯器」(Mac) → 貼上 → 儲存時檔名取 index.html → 雙擊用瀏覽器打開。

⭐ 最推薦的方式
🌐

方法二：CodePen 線上預覽

打開 codepen.io → 在 HTML 欄位貼上程式碼 → 右下方馬上就能看到預覽。不需要存檔，直接在瀏覽器看。

⚡

方法三：VS Code 預覽

免費下載 VS Code 編輯器 → 安裝「Live Server」外掛 → 打開 .html 檔案後按 Go Live，每次儲存後自動更新預覽。（適合常常需要修改的人）

📁
圖片放哪裡？如果網頁裡有圖片，記得把圖片檔案和 .html 檔案放在同一個資料夾裡，這樣圖片才能正常顯示。
☁️
想讓別人也能用網址連到你的網站？那就需要把檔案「上架」到網路上，這個步驟叫做「部署」或「託管」，可以使用 GitHub Pages、Netlify 等免費服務。這是另一個主題，我們之後另外教。
常見問題

新手最常問的幾個問題

Claude 說的話我看不懂怎麼辦？ ⌄
完全沒關係！Claude 給你程式碼的時候，你不需要看懂它。把整段複製下來，存成 .html 檔案，瀏覽器打開就能看到網頁。如果有部分不滿意，直接用中文跟 Claude 說「我想改XXX」就好。
可以一直修改嗎？要重新開始嗎？ ⌄
不用重新開始！在同一個對話裡繼續說你想改的東西，例如「把背景換成白色」、「按鈕改圓角」、「導覽列加上關於我這個連結」，Claude 會幫你修改，然後給你新的完整程式碼。
Claude 給的程式碼哪部分要複製？ ⌄
找到以 <!DOCTYPE html> 開頭的那一段，一直到最後的 </html>，把這整段都複製起來，這就是完整的網頁檔案內容。
圖片不顯示、破圖怎麼辦？ ⌄
最常見的原因是：① 圖檔和 .html 不在同一個資料夾，② 圖片的檔名打錯了（大小寫也要完全一樣）。把圖片移到和 html 同一個資料夾，然後確認提示詞裡填的檔名完全正確即可。
能不能用中文以外的語言做網頁？ ⌄
當然可以！你跟 Claude 說幾種語言、要幾個語言版本都行。只要在提示詞裡說清楚，例如「網頁內容用英文呈現」或「需要中英文切換功能」，Claude 都能處理。
