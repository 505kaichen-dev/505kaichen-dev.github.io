三、使用方式
----------------------------------------------------------------
1. 直接開啟：
   用瀏覽器開啟 index.html 即可，無需伺服器。
   本機路徑例：file:///C:/Users/kai128/storage-wiki/index.html

2. 部署至 GitHub Pages（建議）：
   a. 建立 GitHub Repository（public 或 private 均可）
   b. 將整個 storage-wiki 資料夾上傳至 repo 根目錄
   c. 進入 Settings → Pages → Source 選 main branch / root
   d. 幾分鐘後即可透過 https://[username].github.io/[repo]/ 存取

3. 快捷鍵：
   / （斜線）：快速聚焦搜尋框
   Esc：關閉搜尋結果
