第一次使用：設定 Google Maps API

為了保護網站額度與你的使用安全，本站不共用公開 API Key。 你建立的金鑰只會儲存在目前瀏覽器中，不會上傳給本站，也不會讓站長看到。 設定一次後，下次使用同一個瀏覽器會自動沿用。

  1. 使用你的 Google 帳號前往 Google Maps Platform，建立一個專案，並依 Google 的指示啟用帳單。實際費用依 Google 當時的方案與你的用量計算。
  2. 開啟「API 和服務 → 程式庫」，搜尋並啟用以下三項：Maps JavaScript API、Places API (New)、Geocoding API。
  3. 前往「API 和服務 → 憑證」，選擇「建立憑證 → API 金鑰」。複製建立好的金鑰，但先不要關閉設定頁。
  4. 點進剛建立的金鑰，將「應用程式限制」選為「網站」，並在允許的網站中加入本站網址：https://505kaichen-dev.github.io/food-spinner/*
  5. 將「API 限制」選為「限制金鑰」，只勾選前面啟用的三項 API，然後按下儲存。這一步可以避免金鑰被拿去使用其他 Google 服務。
  6. 回到這個頁面，將金鑰貼入上方欄位，再按「驗證 API Key」。看到綠色 LIVE 就完成了；Google 的新設定偶爾需要等待約 5 分鐘才會生效。

🔐 共用電腦使用完畢時，請按「清除 Key」。若驗證失敗，畫面會告訴你缺少哪一項權限；通常不需要開啟開發者工具。
