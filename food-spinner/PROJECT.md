# Food Spinner

## 專案概要

- 正式入口：`index.html`
- 首頁分類：作品分享（`works`）
- 首頁卡片序號：`014`
- 封面圖片：`image/cover.png`（1200×630）
- 公開網址：`https://505kaichen-dev.github.io/food-spinner/`

## 功能

- 依地址或 GPS 搜尋附近餐飲店家。
- 早餐／中餐／晚餐切換與距離篩選。
- 僅顯示營業中、排除便利商店。
- 篩選後不足時自動補搜，並以 Place ID 去除重複。
- 轉盤隨機抽選，結果可直接開啟 Google Maps。
- API Key 驗證、設定教學與清除功能。
