# bopomofo 例字與注音校對紀錄

更新日期：2026-06-27

## 校對範圍

- `learn.html`：37 個注音符號、每個符號 3 個例字，共 111 筆。
- `chart.html`：37 個符號的單字示例與 Emoji。
- 檢查項目：完整詞語注音、聲調、輕聲、目標符號、詞義與 Emoji 配對。

## 主要修正

- 多音節詞改為完整注音，例如「爸爸」由 `ㄅㄚˋ` 改為 `ㄅㄚˋ ˙ㄅㄚ`。
- 補上輕聲與遺漏聲調，例如杯子、帽子、弟弟、奶奶、早上、晚上、耳朵等。
- 修正錯音，例如「沙灘」原標成 `ㄙㄚ ㄊㄢ`，並改用真正包含 ㄙ 的「雨傘」。
- 修正不包含目標韻母的例字：ㄛ 不再使用「鴿子／河」。
- ㄝ 的三個重複「耶」改為「姐姐、蝴蝶、鞋子」。
- ㄟ 移除多音且容易混淆的「欸」，改用「飛機」。
- 修正詞義不符的 Emoji，例如爸爸、婆婆、弟弟、奶奶、啊、哦、嗯、兒子、而且等。
- `chart.html` 同步修正 ㄋ、ㄌ、ㄚ、ㄛ、ㄝ、ㄟ、ㄣ、ㄤ 等示例與 Emoji。

## 驗證結果

- 111 筆例字皆包含所屬的目標注音符號。
- 111 筆皆為互不重複的三個例字組合。
- 教育部資料庫可直接比對的 110 筆，完整注音均一致。
- 「烏龜」未以完全相同詞目命中下載資料，保留通行讀音 `ㄨ ㄍㄨㄟ`。
- 37 個總表示例均確認包含對應符號。

## 例字直式注音版面

- 參考《翻轉教育》注音字型文章中「國字右側逐字直排注音」的呈現方式。
- `learn.html` 的例字卡已由橫排注音改為每個國字各自配對直式注音。
- 輕聲點獨立置於注音上方；二、三、四聲調號置於注音右側；第一聲不加調號。
- 沿用已校對的 111 筆例字資料，不依賴裝置預裝特殊字型，手機與平板均可顯示。
- 保留完整的無障礙文字標示，點擊例字朗讀功能不變。
- JavaScript 語法檢查通過。
