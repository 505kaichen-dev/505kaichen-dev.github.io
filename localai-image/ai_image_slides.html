Chapter 01 — 基本概念

四個核心名詞

搞懂這四個，後面所有東西都容易理解。

🧠

模型（Model）

AI 的「大腦」，決定它能畫什麼、畫得多好。不同模型擅長不同風格，就像不同畫家有不同畫風。

💬

提示詞（Prompt）

用文字告訴 AI 你想要什麼。寫得越清楚，結果越接近你想要的。這是最重要的技能。

🎛

LoRA

風格外掛。裝上 LoRA 可以讓 AI 生成特定風格的圖，例如日系插畫、像素風、水彩風。

🖼

VRAM（顯示記憶體）

顯示卡的工作記憶體。你的 RTX 4060 有 8GB，足夠跑大部分圖像生成任務。

💡
簡單理解：模型是畫家本人，LoRA 是他的畫風設定，Prompt 是你給他的任務，VRAM 是他的工作台大小。

Chapter 02 — 安裝環境

四步驟完成環境建置

整個過程約 30–60 分鐘，大部分是等待下載。硬碟需要至少 50GB 空閒空間。

1

下載並安裝 Pinokio

前往 pinokio.computer 下載 Windows 版本 → 安裝時選空間大的硬碟，不要裝在 C 槽根目錄

2

在 Pinokio 安裝 Stable Diffusion WebUI Forge

點左側「Explore」→ 搜尋「Stable Diffusion WebUI Forge」→ Download → Install → 等待完成

3

前往 Civitai 下載模型

civitai.com → 搜尋「DreamShaper XL」→ 下載 .safetensors 檔案 → 放入 sd-forge / models / Stable-diffusion /

4

啟動並測試出圖

Pinokio 點「Start」→ 瀏覽器開啟介面（localhost:7860）→ Prompt 輸入「a cup of coffee」→ Generate → 出現圖片即成功

Chapter 05 — 提示詞

Prompt 的基本結構

把你的描述分成這幾個區塊，依序填入

Prompt 結構說明
// 1. 主體描述（最重要，寫最清楚） 主體：a young woman holding a coffee cup, smiling // 2. 風格設定 風格：commercial photography, studio lighting, professional // 3. 畫面品質 品質：high quality, 8k, sharp focus, masterpiece // 4. 背景 / 環境 環境：white background, clean minimal // 5. Negative Prompt（告訴 AI 不要什麼） 不要：blurry, ugly, deformed hands, watermark, text, low quality
⚠️
重要：Prompt 用英文寫效果最好。先用中文想好描述，再貼給 ChatGPT 翻成英文 Prompt，這樣兩個工具都用到。

Chapter 05 — 提示詞

實際範例：日系插畫社群配圖

把這兩段直接複製進 Forge 就可以開始測試

Positive Prompt ✅
cute anime girl, sitting at desk, working on laptop, cozy room, warm lighting, autumn mood, chibi style, soft colors, high quality illustration, pastel color palette, window with sunlight, potted plants, coffee cup on desk
Negative Prompt ❌（幾乎通用，固定用這段）
blurry, ugly, deformed, extra fingers, bad anatomy, watermark, text, signature, cropped, worst quality, low quality, jpeg artifacts, distorted face, nsfw
💡
Negative Prompt 這段存起來，以後每次都直接貼，不用重寫。

Chapter 06 — 參數設定

四個關鍵參數

照以下建議值設定，不要亂改其他東西。

尺寸

圖片尺寸（Width × Height）

IG 方形：1024×1024
IG 限時：768×1344（9:16）
FB 橫幅：1344×768（16:9）
文章封面：1216×832

步數

Sampling Steps（取樣步數）

快速測試：20 步（確認方向）
正式輸出：28–35 步（品質好）
不需要超過 40 步，效果差異極小

CFG

CFG Scale（提示詞服從度）

建議固定設 7，這是最安全的通用值。太低 AI 亂發揮；太高圖片過度飽和。

批次

Batch Count（批次數量）

測試 Prompt 時：設 4 張，快速看方向
確認方向後：設 1 張，精細輸出

Chapter 07 — 工作流程

五步驟標準流程

照這個順序走，不容易卡關。

1

確認用途和風格

這張圖要放在哪？（IG / Facebook / 文章封面）要什麼感覺？（寫實 / 插畫 / 溫暖 / 簡約）

2

選對模型和尺寸

插畫選 AnythingXL，寫實選 RealVisXL，不確定就用 DreamShaper XL。尺寸依據用途設定。

3

寫 Prompt → 設 4 張批次快速測試

Steps 設 20，Batch 設 4，先看方向對不對。找到喜歡的方向後，記下那張的 Seed 數字。

4

調整 Prompt → 正式輸出

Steps 改 30，Batch 改 1，鎖定 Seed，微調 Prompt 細節，輸出最終版本。

5

後製（選擇性）

用 Canva 加文字和 Logo。AI 負責生成畫面，排版文字用熟悉的工具比較快。

Chapter 09 — 常見問題 (1/2)

Q&A

Q

生成的圖畫質很差、很模糊？

① Negative Prompt 沒填 → 加入 blurry, low quality, worst quality
② Steps 太少 → 改成 28–35
③ 試試換不同模型

Q

人物的手畫得很奇怪、手指數量不對？

① Negative Prompt 加 extra fingers, deformed hands, bad anatomy
② 構圖讓手不是焦點，或讓手持物品遮擋
③ 多生成幾張挑比較好的

Q

顯示卡記憶體不足（CUDA out of memory）？

① 縮小圖片尺寸從 1024 降到 896 或 768
② Batch Count 改成 1
③ Forge 啟動設定加入 --medvram
④ 關掉其他佔 VRAM 的程式

23
Chapter 09 — 常見問題 (2/2)

Q&A

Q

每次生成的圖都不一樣，怎麼鎖定喜歡的結果？

每張圖都有一個 Seed 數字，記下這個數字，下次填入同樣的 Seed，在其他條件不變的情況下可以重現相近的結果。在 Forge 介面找到「Seed」欄位，把 -1 改成記下的數字。

Q

可以生成有中文字或 Logo 的圖嗎？

目前 AI 圖像生成在文字上非常不準確，不要期待 AI 生成精確的文字。正確做法：AI 生成沒有文字的底圖，Canva / Photoshop 後製加上文字和 Logo。

Q

同一個人物可以在不同場景重複出現嗎？

這需要 IP-Adapter 或 角色 LoRA 技術，屬於進階操作。新手先從單張圖練習，熟練後再學角色一致性工作流。
